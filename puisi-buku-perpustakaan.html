6. jika rata rata banyak buku di perpustakaan 1325 buku, berapakah banyak majalah ywng ada di perpustakaan?


Jika rata rata banyak buku di perpustakaan 1.325 buku, berapakah banyak majalah yang ada di perpustakaan adalah 800 buku.


Pembahasan :

Dari soal  (Gambar yang terlempir), diketahui jumlah buku-buku perpustakaan sebagai berikut :

Buku Pelajaran : 1.200

Buku Cerita : 2.400

Majalah : x

Ensiklopedia : 900

Rata-Rata Banyak Buku = 1.325


Rata - Rata = (Jumlah Buku) / (Jumlah Jenis Buku)

1.325 = (1.200 + 2.400 + x + 900) / (4)

1.325 = (4.500 + x) / (4)

1.325 × 4 = 4.500 + x

5.300 = 4.500 + x

x = 5.300 - 4500

x = 800 buku

Jadi banyak majalah yang ada di perpustakaan adalah 800 buku.
