6. hai aku lagi ujian mohon doa ya temen" ini quis tuk kalian sory plit poin2⁹+4⁵ =​


Jawaban:

2⁹+4⁵ = 1536✔️

2⁹ =

~ 2×2×2×2×2×2×2×2×2

~ 4×2×2×2×2×2×2×2

~ 8×2×2×2×2×2×2

~ 16×2×2×2×2×2

~ 32×2×2×2×2

~ 64×2×2×2

~ 128×2×2

~ 256×2

~ 512✔️

4⁵ =

~ 4×4×4×4×4

~ 16×4×4×4

~ 64×4×4

~ 256×4

~ 1024✔️

2⁹+4⁵ = 512+1024 = 1536✔️

2. a=98 b= -6
a. rumus suku ke n = a + (n-1) b
= 98 + (n-1) -6
= 98 - 6n + 6 = 104 - 6n
b. suku ke 25 = 98 + (25-1) -6
= 98 + 24.-6
= 98 - 144 = -46

3. U1 = 3.1 - 2 = 3 - 2 = 1
U2 = 3.2 -2 = 6-2 = 4
U3 = 3.3 - 2 = 9 -2 = 7
U4 = 3.4 - 2 = 12-2 = 10
U5 = 3.5 - 2 = 15-2 = 13

U7 = 3.7 - 2 = 21- 2 = 19

4. rumus suku ke n = 3 + n^2
suku ke 10 = 3 + 10^2 = 3 + 100 = 103

5. a=1 b=4
69 = a + (n-1) b
69 = 1 + (n-1) 4
69 = 1 + 4n -4
4n = 69 - 1 + 4 = 72
n = 72/4 = 18
maka banyak suku (n) = 18
