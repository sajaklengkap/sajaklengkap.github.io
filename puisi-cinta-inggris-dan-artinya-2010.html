28. hitunglah nilai dari 2012^2+2(2012)(2010)+2010^2 / 2012^2-2010^2= ("^" artinya "pangkat" dan "/" artinya ''per'')


2012² + 2 (2012)(2010 + 2010² / 2012² - 2010²
= (2012 + 2010)(2012 + 2010) / (2012 - 2010)²
= (2012 + 2010)² / - (2012 + 2010)²
= -1
