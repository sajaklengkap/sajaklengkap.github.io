10. pak joko memasang genting, genting disusun baris paling atas 20 genting baris paling bawah 34 genting dan susunan genting terdiri atas 15 baris berapa banyak genting di atap


pemasangan genting ini mengikuti barisan aritmatika, maka bisa diuraikan menjadi :

a = suku pertama = baris pertama = 20 genting
U15 = suku ke-15 = baris akhir = 34 genting
n = 15 baris

U15 = 34
35 = a + 14b
34 = 20 + 14b
34 - 20 = 14b
14 = 14b
b = 1

S15 = n/2 (a + Un)
= 15/2 (a + U15)
= 15/2 (20 + 34)
= 15/2 (54)
= 15 (27)
= 405

maka jumlah genting di atas ada 405 genting.
