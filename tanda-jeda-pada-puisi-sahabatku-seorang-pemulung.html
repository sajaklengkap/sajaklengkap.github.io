Sahabatku Seorang Pemulung
Ciptaan : D. Karitas

Tangan-tangan kecilmu begitu lincah//
Mengais / dan memilah sampah rumah //
Kamu // mencari plastik //
Bahkan kertas dan kardus //

Berbekal karung dan tongkat kecil //
Kamu // mencari dan menemukan //
Barang-barang yang telah dibuang pemiliknya //
Untuk // ditukarkan kepada para pengepul //

Aku mengagumimu, // sahabatku //
Kamu // tidak pernah menyerah
Kamu // tidak pernah mengeluh //
Kamu // selalu bersemangat //

Semoga kamu / dapat menggapai cita-citamu //
Seperti aku pun // berjuang untuk itu //
Mari kita saling mendukung //
Untuk menjadi yang kita mau ///

Pemenggalan kalimat harus diperhatikan dalam membaca puisi agar puisi tersebut enak didengar dan mudah dipahami isinya.
Tanda jeda dalam membaca puisi:
1. / = berhenti sejenak untuk mengambil napas, biasanya karena ada koma atau di tengah baris.
2. // = berhenti agak lama, biasanya adanya koma di akhir baris atau ganti baris
3. /// = berhenti lama sekali, biasanya pada akhir baris terakhir atau akhir dari puisi.

5. Rima dalam puisi sahabatku seorang pemulung Tema dalam puisi sahabatku seorang pemulung


Rima dan tema adalah dua unsur intrinsik yang terdapat dalam sebuah puisi, yaitu unsur yang berasal dari dalam puisi. Rima dapat diartikan sebagai perulangan bunyi yang terjadi pada baris-baris puisi setiap baitnya. Sementara tema merupakan gagasan besar yang mendasari penulisan puisi.


Pembahasan

Pada kesempatan ini, soal menyajikan kita dengan sebuah puisi yang berjudul "Sahabatku Seorang Pemulung". Kemudian, kita diminta untuk menentukan rima dan tema yang digunakan pada puisi tersebut. Berikut kakak akan mencoba menjawab pertanyaan tersebut satu per satu.


PUISI RUJUKAN

Sahabatku Seorang Pemulung.

Ciptaan: D. Karitas

Tangan-tangan kecilmu begitu lincah

Mengais dan memilah sampah rumah

Kamu mencari plastik,

Bahkan kertas dan kardus.

Berbekal karung dan tongkat kecil

Kamu mencari dan menemukan

Barang-barang yang telah dibuang pemiliknya

Untuk ditukarkan kepada para pengepul.

Aku mengagumimu, sahabatku

Kamu tidak pernah menyerah

Kamu tidak pernah mengeluh

Kamu selalu bersemangat.

Semoga kamu dapat mencapai cita-citamu

Seperti aku pun berjuang untuk itu

Mari kita saling mendukung

Untuk menjadi yang kita mau.
