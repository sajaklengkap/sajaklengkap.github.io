Jawaban:

Rp565.000,00

Penjelasan dengan langkah-langkah:

Diketahui:

Diskon 20% = Baju

Diskon 25% = Tas

Diskon 30% = Sepatu

Baju (awal) = 300.000

Tas (awal) = 200.000

Sepatu (awal) = 250.000

Ditanya:

Harga akhir?

Jawab:

• Harga baju = Harga awal - Diskon

= 300.000 - (20% × 300.000)

= 300.000 - (20/100 × 300.000)

= 300.000 - 60.000

= 240.000

• Harga tas = Harga awal - Diskon

= 200.000 - (25% × 200.000)

= 200.000 - (25/100 × 200.000)

= 200.000 - 50.000

= 150.000

• Harga sepatu = Harga awal - Diskon

= 250.000 - (30% × 250.000)

= 250.000 - (30/100 × 250.000)

= 250.000 - 75.000

= 175.000

☞ Harga keseluruhan

= Harga baju + Harga tas + Harga sepatu

= 240.000 + 150.000 + 175.000

= 565.000

Jadi, bu Riska harus membayar Rp565.000,00

kelas : X SMA
mapel : matematika
kategori : pertidaksamaan linier dua variabel
kata kunci : soal cerita

kode : 10.2.4 [matematika SMA kelas X bab 4 pertidaksamaan linier dua variabel]

Pembahasan:

untuk menyelesaikan soal seperti ini kita buat terlebih dahulu kalimat matematikanya, dan kita tentukan nilai untuk variabel x dan nilai untuk variabel y.
pertidaksamaan linier dua variabel adalah kalimat terbuka yang menggunakan 2 variabel, dan masing-masing variabel berderajat 1, serta menggunakan tanda penghubung <, >, ≤, ≥.
bentuk pertidaksamaan linier dapat ditulis:
ax + by ≤ c
ax + by ≥ c
ax + by < c
ax + by > c
dan hasilnya berbeda dibandingkan dengan persamaan, jika dalam persamaan hasilnya berupa himpunan pasangan titik-titik, namun jika pertidaksamaan linier dua variabel hasilnya berupa daerah arsiran

dalam soal, kita buat pertidaksamaannya terlebih dahulu:
baju model 1 = x
baju model 2 = y

                                polos    corak    harga jual
baju model 1 = x        2           1           350.000
baju model 2 = y       1            2          200.000
    total                       8           10
dari bagan ini dapat kita buat pertidak

2x + y ≤ 8
x + 2y ≤ 10
z = 350.000x + 200.000y
ditanya hasil penjualan maksimum dari penjualan kedua model baju = ...?

jawab:
kita buat nol fungsi terlebih dahulu
2x + y ≤ 8
memotong sumbu x di titik y = 0
2x + y = 8
2x + 0 = 8
2x = 8
x = 8/2
x = 4
memotong sumbu y di titik x = 0
2x + y = 8
0 + y = 8
y = 8

x + 2y ≤ 10
memotong sumbu x apabila y = 0
x + 2y = 10
x + 0 = 10
x = 10
memotong sumbu y apabila x = 0
x + 2y = 10
0 + 2y = 10
y = 10/2
y = 5

langkah selanjutnya kita eliminasi
2x + y = 8   I*1I   2x + y = 8
x + 2y = 10 I*2I  2x + 4y = 20
                          --------------------- -
                                -3y         = -12
                                    y         = -12/-3
                                    y         = 4
x + 2y = 10
x + 2(4)  = 10
x + 8 = 10
x = 10 - 8
x = 2

kita tentukan titik uji yang memenuhi
kita dapat 3 titik uji, perhatikan gambar pada lampiran
1) (0, 5)
2) (2, 4)
3) (4, 0)
subsitusikan titik uji pada nilai z = 350.000x + 200.000y
1) titik uji pertama (0, 5)
    z = 350.000x + 200.000y
      = 350.000(0) + 200.000(5)
      = 0 + 10.000.000
      = 1.000.000
2) titik uji kedua (2, 4)
    z = 350.000x + 200.000y
       = 350.000(2) + 200.000(4)
       = 700.000 + 800.000
      = 1.500.000
3) titik uji ketiga (4, 0)
    z = 350.000x + 200.000y
      = 350.000(4) + 200.000(0)
      = 1.400.000 + 0
      = 1.400.000

hasil penjualan maksimum dari penjualan 2 model baju yaitu 1.500.000 dengan menjual 2 baju model 1  dan 4 baju model 2

9. Sebuah butik memberikan diskon 15% untuk baju dan 12% untuk rok. Jika Annisa membeli satu baju dengan harga Rp125.000 dan satu rok dengan harga Rp85.000, maka uang yang harus dibayar Annisa adalaha) Rp181.050,00b) Rp191.250,00c) Rp199.800,00d) Rp210.000,00​


Menentukan harga yang sudah didiskon

Baju = harga baju - diskon

Baju = Rp. 125.000,00 - 15%

Baju = Rp. 106.250,00

Rok = Harga rok - diskon

Rok = Rp. 85.000,00 - 12%

Rok = Rp. 74.800,00

Menentukan uang yang harus dibayar

Uang = Harga baju + Harga rok ( Sudah Diskon )

Uang = Rp. 106.250,00 + Rp. 74.800,00

Uang = Rp. 181.050,00.

Jadi, uang yang harus dibayar oleh Anissa adalah Rp. 181.050,00.

Jawab:

*harga diskon baju = 15/100 x 125.000

                               = 18.750

harga baju setelah diskon = 125.000-18.750

                                            =106.250

*harga diskon rok = 12/100 x 85.000

                              = 10.200

harga rok setelah diskon = 85.000-10.200

                                          = 74.800

jadi, uang yang harus dibayarkan oleh annisa jika ia membeli satu baju dan rok adalah 106.250 + 74.800= 181.050,00 (a)

pertidaksamaan linier dua variabel adalah kalimat terbuka yang menggunakan 2 variabel, dan masing-masing variabel berderajat 1, serta menggunakan tanda penghubung <, >, ≤, ≥.
bentuk pertidaksamaan linier dapat ditulis:
ax + by ≤ c
ax + by ≥ c
ax + by < c
ax + by > c
dan hasilnya berbeda dibandingkan dengan persamaan, jika dalam persamaan hasilnya berupa himpunan pasangan titik-titik, namun jika pertidaksamaan linier dua variabel hasilnya berupa daerah arsiran
