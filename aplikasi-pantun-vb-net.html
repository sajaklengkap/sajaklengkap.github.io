1. Sebutkan struktur program dalam VB net​


Jawaban:

Sebelum kita mempelajari blok bangunan dasar bahasa pemrograman VB.Net, mari kita lihat struktur program VB.Net yang minimal sehingga kita dapat menganggapnya sebagai referensi dalam bab-bab yang akan datang.

Program VB.Net pada dasarnya terdiri dari beberapa bagian berikut:

Deklarasi namespace

Kelas atau modul

Satu atau lebih prosedur

Variabel

Prosedur utama

Pernyataan & Ekspresi

Komentar

Mari kita lihat sebuah kode sederhana yang akan mencetak kata-kata "Hello World":

1

2

3

4

5

6

7

8

Imports System

Module Module1

'This program will display Hello World

Sub Main()

Console.WriteLine("Hello World")

Console.ReadKey()

End Sub

End Module

Bila kode diatas dikompilasi dan dieksekusi, maka menghasilkan hasil sebagai berikut:

Hello, World !

Mari kita lihat berbagai bagian dari program di atas:

Baris pertama dari program Imports System digunakan untuk memasukkan namespace System dalam program ini.

Baris berikutnya memiliki deklarasi Modul, Modul Modul1. VB.Net benar-benar berorientasi objek, jadi setiap program harus berisi modul kelas yang berisi data dan prosedur yang digunakan program Anda.

Kelas atau Modul umumnya mengandung lebih dari satu prosedur. Prosedur berisi kode yang dapat dieksekusi, atau dengan kata lain, mereka mendefinisikan perilaku kelas. Prosedur bisa berupa salah satu dari berikut ini:

Function

Sub

Operator

Set

AddHandler

RemoveHandler

RaiseEvent

Baris berikutnya ('Program ini) akan diabaikan oleh compiler dan telah ditambahkan komentar tambahan di program ini.

Baris berikutnya mendefinisikan prosedur utama, yang merupakan titik masuk untuk semua program VB.Net. Prosedur Utama menyatakan apa yang akan dilakukan modul atau kelas saat dijalankan.

Prosedur Utama menentukan perilakunya dengan pernyataan tersebut.

Console.WriteLine ("Hallo World") WriteLine adalah metode kelas Console yang didefinisikan dalam namespace Sistem. Pernyataan ini menyebabkan pesan "Hallo, World!" Untuk ditampilkan di layar. Baris terakhir Console.ReadKey () adalah untuk Pengguna VS.NET. Ini akan mencegah layar agar tidak berjalan dan menutup dengan cepat saat program diluncurkan dari Visual Studio.NET.

Compile & Execute Program VB.Net:

Jika Anda menggunakan Visual Studio.Net IDE, lakukan langkah-langkah berikut:

Mulai Visual Studio.

Pada bilah menu, pilih File, New, Project.

Pilih Visual Basic From Template

Pilih Console Application

Tentukan nama dan lokasi proyek Anda dengan menggunakan tombol Browse, kemudian pilih tombol OK

Proyek baru muncul di Solution Explorer.

Tulis kode di Editor Kode.

Klik tombol Run atau tombol F5 untuk menjalankan proyek. Jendela Command Prompt muncul yang berisi baris Hello World.

Anda dapat mengkompilasi sebuah program VB.Net dengan menggunakan command line daripada Visual Studio IDE:

Buka editor teks dan tambahkan kode yang disebutkan di atas.

Simpan file sebagai helloworld.vb

Buka alat command prompt dan masuk ke direktori tempat Anda menyimpan file.

Ketik vbc helloworld.vb dan tekan enter untuk mengkompilasi kode Anda.

Jika tidak ada kesalahan dalam kode Anda, command prompt akan membawa Anda ke baris berikutnya dan akan menghasilkan file eksekusi helloworld.exe.

Selanjutnya, ketik helloworld untuk menjalankan program anda.

Anda akan dapat melihat "Hello World" yang tercetak di layar.

vb .net adalah

Library-nya yang lengkap justru membuat waktu komputasi jadi lebih lama.

Fluida dinamis merupakan ilmu terkait dengan fluida yang bergerak, meliputi hukum kontinuitas, bernaouli dan toricelli. Pada fluida ini erat kaitannya dengan debit air. Debit merupakan jumlah air yang mengalir dalam satuan waktu dan memenuhi persamaan  

Q=V/t=Av  

dimana debit akan konstan memenuhi persamaan kontinuitas pada pipa yang sama  

A1v1=A2v2  

Pada fluida dinamis hukum bernauli sangat diperlukan dalam hampir seluruh perhitungan fluida dinamis. Hukum Bernoulli merupakan hukum yang berlandaskan kekekalan energi per unit volume pada aliran fluida, pada hukum ini bernauli menggunakan kekekalan energi pada dua tempat sehingga

[tex]Em_1 = Em_2\\P_1 + Ek_1 + Ep_1 = P_2 + Ek_2 + Ep_2\\P_1 + \frac{1}{2} \rho v_1^2 + \rho g h_1 = P_2 + \frac{1}{2} \rho v_2^2 + \rho g h_2[/tex]

dimana:  

P adalah tekanan (Pa)  

[tex]\rho[/tex] adalah massa jenis fluida (kg/m3)  

g adalah percepatan gravitasi (9,8 m/s2)  

h adalah ketinggian air (m)  

v adalah kecepatan aliran fluida (m/s)

Pada air yang mengalir pada tandon air dapat digunakan pula hukum bernauli sehingga kecepatan air yang mengalis dibawah akan sama dengan

[tex]\frac{1}{2} \rho v^2 = \rho g h\\v^2 = 2 g h\\v = \sqrt{2gh}[/tex]

Pembahasan  

Perhatikan gambar, gaya angkat pesawat nilainya

[tex]F=0,5\rho (v_a^2-v_b^2)A[/tex]

karena soalya Pg pilih aja yang selisihnya paling besar, yakni D. . va = 30 vb = 10

kita cek

[tex]F=0,5\rho (v_a^2-v_b^2)A\\F=0,5*1(30^2-10^2)200\\F=8000\; N[/tex]

dan gaya beratnya 7500 N,

Andi dan budi melakukan percobaan dengan melemparkan bola.Andi melemparkan bola ke atas dengan kelajuan va dan budi melemparkan ke bawah dengan kecepatan vb jarak kedua bola tersebut 2h.Jika percepatan gravitasi dinyatakan dengan g, persamaan jarak yang ditempuh benda b adalah....A. HB= 2h(vA+vB^2+gh)/(vA+bab)^2B. HB =2h(vA+vB^2-gh)/(vA+bab)^2C. HB =2h(vA.VB+vB^2+gh)/(vA+bab)^2D. HB =2h(vAvB+vB^2+gh)/(vA-vB)^2E. HB= 2h(vAvB-vB^2+gh)/(vA-vB)^2


Jawaban:

Jarak yang ditempuh oleh B sehingga bertemu dengan A adalah [tex]h_b=\frac{2h(v_av_b+v_b^2+gh)}{(v_a+v_b)^2}[/tex]

Penjelasan:

Untuk menyelesaikan soal diatas dapat digunakan persamaan GLBB

Jarak total keduanya adalah 2h sehingga

[tex]h_a+h_b=2h\\[/tex]

Persamaan gerak benda a adalah

[tex]h_a=v_at-0,5gt^2\\[/tex]

Persamaan gerak benda b adalah

[tex]h_b=v_bt+0,5gt^2[/tex]

Sehingga

[tex]h_a+h_b=v_at+v_bt\\2h=(v_a+v_b)t\\t=\frac{2h}{v_a+v_b}[/tex]

Dengan demikian jarak b adalah

[tex]h_b=v_bt+0,5gt^2\\h_b=v_b\frac{2h}{v_a+v_b} +0,5g*(\frac{2h}{v_a+v_b} )^2\\h_b=v_b\frac{2h(v_a+v_b)}{(v_a+v_b)^2} +0,5g*(\frac{2h}{v_a+v_b} )^2\\h_b=\frac{2h(v_av_b+v_b^2)+2gh^2}{(v_a+v_b)^2} \\h_b=\frac{2h(v_av_b+v_b^2+gh)}{(v_a+v_b)^2}[/tex]
