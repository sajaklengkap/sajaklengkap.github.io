3. Kedai kopi menjual seratus cangkir kopi pada hari Sabtu, dan dua ratus cangkir pada hari Minggu. Kedai kopi ini selalu menjual dengan jumlah yang sama setiap minggunya di bulan ini. Berapa cangkir kopi yang terjual di kedai ini dalam satu bulan?​


Jawab:

Penjelasan dengan langkah-langkah:

Dalam satu akhir pekan (Sabtu dan Minggu), kedai kopi menjual 100 + 200 = 300 cangkir kopi.

Karena kedai kopi ini menjual dengan jumlah yang sama setiap akhir pekan dalam sebulan, maka kita dapat mengalikan jumlah penjualan akhir pekan (300 cangkir) dengan jumlah akhir pekan dalam sebulan. Biasanya, sebulan memiliki sekitar 4 akhir pekan (8 hari akhir pekan), tetapi untuk menghindari asumsi yang salah karena beberapa bulan memiliki akhir pekan lebih banyak atau lebih sedikit, mari gunakan 4 akhir pekan sebagai contoh.

Jadi, jumlah total cangkir kopi yang terjual dalam satu bulan adalah:

300 cangkir/akhir pekan x 4 akhir pekan/bulan = 1200 cangkir/bulan.

Jadi, kedai kopi ini menjual 1200 cangkir kopi dalam satu bulan.

Kelas : 9
Materi : Bab 3 - Barisan dan Deret
Kata Kunci : Deret Aritmatika
Kode Kategorisasi : 9.2.3 ( Matematika Kelas 9 Bab 3 - Barisan dan Deret )
————————————————————
3 Minggu = 21 Hari ...

Un = 20 + 15n

U1 = 20 + 15 ( 1 )
U1 = 20 + 15
U1 = 35 ...

U2 = 20 + 15 ( 2 )
U2 = 20 + 30
U2 = 50 ...

U3 = 20 + 15 ( 3 )
U3 = 20 + 45
U3 = 65 ...

b = U2 - U1 atau U3 - U2
b = 50 - 35
b = 15 ...

Sn = n / 2 ( 2a + ( n - 1 ) b )
S21 = 21 / 2 ( 2 ( 35 ) + ( 21 - 1 ) 15 )
S21 = 10,5 ( 70 + ( 20 ) 15 )
S21 = 10,5 ( 70 + 300 )
S21 = 10,5 ( 370 )
S21 = 3.885 kg KOPI ..... ✔️

Diketahui, Un = 15n + 20
a = 15(1) + 20 = 35
Ditanya : S3 minggu = S21 hari = ?

Sn = n/2 (a + Un)
S21 = 21/2 (35 + U21)
S21 = 21/2 (35 + (20 + 15(21)))
S21 = 21/2 (35 + 335)
S21 = 21/2 (370)
S21 = 3885

Jadi, massa total kopi yang dipetik selama 3 minggu pertama adalah 3885 kg.Mapel : Matematika
Kelas : 9
Materi : Bab 3 - Barisan dan Deret
Kata Kunci : Deret Aritmatika
Kode Kategorisasi : 9.2.3 ( Matematika Kelas 9 Bab 3 - Barisan dan Deret )
————————————————————
3 Minggu = 21 Hari ...

Un = 20 + 15n

U1 = 20 + 15 ( 1 )
U1 = 20 + 15
U1 = 35 ...

U2 = 20 + 15 ( 2 )
U2 = 20 + 30
U2 = 50 ...

U3 = 20 + 15 ( 3 )
U3 = 20 + 45
U3 = 65 ...

b = U2 - U1 atau U3 - U2
b = 50 - 35
b = 15 ...

Sn = n / 2 ( 2a + ( n - 1 ) b )
S21 = 21 / 2 ( 2 ( 35 ) + ( 21 - 1 ) 15 )
S21 = 10,5 ( 70 + ( 20 ) 15 )
S21 = 10,5 ( 70 + 300 )
S21 = 10,5 ( 370 )
S21 = 3.885 kg KOPI ..... ✔️

Biaya tetap mingguan stan kopi adalah Rp2.000.000

Biaya variabel

2000 gelas 2500x2000 = 5.000.000

2100 gelas 2500x2100 = 5.250.000

2200 gelas 2500x 2200= 5.500.000

Jika 2100, maka = Rp 952 berasal dari 3452-2500
