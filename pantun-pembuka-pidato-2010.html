25. sebutkan beberapa cara membuka microsoft word 2010


a. Melalui Start Menu
  Ø  Klik Start menu yang ada di pojok bawah kiri pada layar monitor
  Ø  Klik programs/all programs
  Ø  Sorot dan klik Microsoft Office
  Ø  Pilih dan klik microsoft office word

b. Melalui Shortcut
  Ø  Sebelumnya kita buat dulu shortcut untuk Ms. Word pada dekstop
  Ø  Arahkan kursor pada shortcut Ms. Word yang sudh dibuat
  Ø  Double klik pada shortcut Ms. Word.
  Ø  Tunggu sampai Ms. Word siap dioperasikan.

c. Menggunakan fasilitas RUN
  Ø  Klik Start menu >> pilih dan klik RUN
  Ø  Muncul dialog RUN >> pilih dan klik Browse
  Ø  Kemudian kita cari letak file yang akan kita buka.
  Ø  Setelah ketemu  klik OK

d. Melalui  Windows Explorer
  Ø  Buka dahulu Windows Explorer, yaitu
  Ø  Klik Tombol Start menu >> klik Accessories >> Windows Explorer
  Ø  Tampilan windows explorer akan muncul
  Ø  Kemudian pilih dan klik file Microsoft Word.

e. Melalui Fasilitas Pencarian
  Ø  Klik Start menu >> klik Search >> pilih All files and folders
  Ø  Pada form isian kita bisa menuliskan nama file yang akan kita buka, misalnya winword.
  Ø  Setelah ditemukan kita klik file tersebut,, tunggu beberapa saat sampai file tersebut terbuka

31. sebutkan dua cara membuka dokumen Microsoft word 2010 yang tersimpan di komputer


1. Lewat perintah Open pada Word, caranya:
- Buka Ms. Word
- Klik Office Button (untuk Word versi 2007) atau ribbon File (untuk Word versi 2010 keatas), lalu klik Open, atau bisa juga dengan menekan kombinasi tombol Ctrl+O pada keyboard, lalu tunggu hingga jendela Open muncul
- Kemudian, pilih direktori/tempat kamu menyimpan dokumen tersebut, kemudian klik/pilih dokumen tersebut & klik Open

2. Cara Langsung, caranya:
- Buka Windows Explorer dengan cara klik kiri Start Menu, kemudian klik Open Windows Explorer, atau bisa juga dengan menekan kombinasi tombol Windows+E pada keyboard & tunggu hingga jendela Explorer muncul
- Setelah Explorer muncul, cari tempat dimana dokumen tersebut kamu simpan (misal di folder My Document), setelah ketemu dokumennya, klik 2x pada dokumen tersebut & tunggu hingga dokumen tersebut muncul
