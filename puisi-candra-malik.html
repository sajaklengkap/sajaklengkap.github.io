= 7 - 5/4 + 7 × 66.000

= 2/11 × 66.000

= 2 × 6.000

= Rp.12.000,-

Jadi, selisih uang Budi dan Candra = Rp.12.000,-

Perbandingan?

Dijawab :

Perbandingan

= 4 : 7

32. Umur Candra 3 tahun yang lalu kurang dari 25 tahun berapakah umur candra sekarang !


Kelas 4 Matematika
Bab Operasi Bilangan

Candra = c

c - 3 = 25
c - 3 + 3 = 25 + 3
c = 28 tahunUmur Candra 3 Tahun Yang Lalu Kurang Dari 25

Ditanya : Umur Candra Sekarang ...?

Jawab :

Cara 1

Candra - 3 = 25
Candra - 3 + 3 = 25 + 3
Candra = 28

Cara 2

Candra - 3 = 25
Candra = 25 + 3 ( - 3 Di balik menjadi + 3 )
Candra = 28

Umur Candra Sekarang Adalah 28 Tahun
