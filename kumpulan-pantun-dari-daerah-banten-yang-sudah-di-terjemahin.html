11. tolong dibantu di terjemahin dong besok mau dikumpulkan​


Ukara ukara ing ndhuwur, menawa dilatenake dadine...

Sunan BonangKyai Abdul KarimSultan TrengganaDewi SintaArab Saudi__________________Pembahasan

Ukara ukara ing ndhuwur nggunakake aksara murda lan aksara swara.

Mangerteni aksara murda lan aksara swara.

1. Aksara murda

Aksara murda

yaiku aksara sesirah kang sinebut huruf kapital. Aksara murda uga digawe kanggo pakurmatan utawa menghormati lan lumrahe kanggo nulis jenenge wong, pangkat lan drajat sarta papan panggonan.

Cacahe aksara murda

➡️ Aksara murda cacahe ana 8 :

Ka = ꦑNa = ꦟTa = ꦡSa = ꦯPa = ꦦNya = ꦘGa = ꦓBa = ꦨ

2. Aksara swara

Mangerteni tegese aksara Swara

Aksara swara yaiku aksara kang digunakake kanggo nulis huruf vokal.

Cacahe aksara swara

➡️ Aksara swara cacahe ana 5 :

A = ꦄI = ꦇU = ꦈE = ꦌO = ꦎ

★ catatan

huruf sing kacethak kandhel lan garis ngisor iku sing nggunakake aksara murda lan aksara swara.

Detail jawabanMapel : Bahasa daerahKelas : 9Materi : Nulis aksara JawaKode : 13

Jawaban:

1. Sunan Bonang

2. Kiai Abdul Karim

3. Sultan Trenggana

4. Déwi Sinta

5. Arab Saudi

Penjelasan:

Nomor 1

• ꦯ (Sa [Aksara Murda]) + ꦸ (Sandhangan U [Suku]) = ꦯꦸ (Su)

• ꦯꦸ (Su) + ꦤ (Na [Aksara Ngelegéna]) = ꦯꦸꦤ (Suna)

• ꦯꦸꦤ (Suna) + ꦺ (Sandhangan É [Taling]) + ꦤ (Na) = ꦯꦸꦤꦤꦺ (Sunané)

• ꦯꦸꦤꦤꦺ (Sunané) + ꧀ꦨ (Pasangan Ba) = ꦯꦸꦤꦤꦺ꧀ꦨ (Sunanbé)

• ꦯꦸꦤꦤꦺ꧀ꦨ (Sunanbe) + ꦴ (Sandhangan Ā [Tarung]) = ꦯꦸꦤꦤꦺ꧀ꦨꦴ (Sunanbo)

• ꦯꦸꦤꦤꦺ꧀ꦨꦴ (Sunanbo) + ꦤ (Na) + ꦁ (Sandhangan Ng [Cecak]) = ꦯꦸꦤꦤꦺ꧀ꦨꦴꦤꦁ (Sunan Bonang)

Nomor 2

• ꦏ (Ka [Aksara Wyanjana]) + ꦾ (Sandhangan Y [Pengkal]) = ꦏꦾ (Kya)

• ꦏꦾ (Kya) + ꦲ (Ha/A [Aksara Ngelegéna]) = ꦏꦾꦲ (Kyaa)

• ꦏꦾꦲ (Kyaa) + ꦶ (Sandhangan I [Wulu]) = ꦏꦾꦲꦶ (Kyai)

• ꦏꦾꦲꦶ (Kyai) + ꦄ (A [Aksara Swara]) + ꦧ (Ba [Aksara Ngelegéna]) = ꦏꦾꦲꦶꦄꦧ (Kyai Aba)

• ꦏꦾꦲꦶꦄꦧ (Kyai Aba) + ꧀ꦢ (Pasangan Da) = ꦏꦾꦲꦶꦄꦧ꧀ꦢ (Kiai Abda)

• ꦏꦾꦲꦶꦄꦧ꧀ꦢ (Kiai Abda) + ꦸ (Sandhangan U [Suku]) = ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸ (Kiai Abdu)

• ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸ (Kiai Abdu) + ꦭ (La [Aksara Wyanjana]) = ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸꦭ (Kiai Abdula)

• ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸꦭ (Kiai Abdula) + ꧀ꦑ (Pasangan Ka [Aksara Murda]) = ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸꦭ꧀ꦑ (Kiai Abdulka)

• ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸꦭ꧀ꦑ (Kiai Abdulka) + ꦫ (Ra [Aksara Ngelegéna]) =ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸꦭ꧀ꦑꦫ (Kiai Abdulkara)

• ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸꦭ꧀ꦑꦫ (Kiai Abdulkara) + ꦶ (Sandhangan I [Wulu]) = ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸꦭ꧀ꦑꦫꦶ (Kiai Abdulkari)

• ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸꦭ꧀ꦑꦫꦶ (Kiai Abdulkari) + ꦩ (Ma [Aksara Wyanjana] + ꧀ (Mati [Pangkon]) = ꦏꦾꦲꦶꦄꦧ꧀ꦢꦸꦭ꧀ꦑꦫꦶꦩ꧀ (Kiai Abdul Karim)

Nomor 3

• ꦯ (Sa [Aksara Murda]) + ꦸ (Sandhangan U [Suku]) = ꦯꦸ (Su)

• ꦯꦸ (Su) + ꦭ (La [Aksara Ngelegéna]) = ꦯꦸꦭ (Sula)

• ꦯꦸꦭ (Sula) + ꧀ꦠ (Pasangan Ta) = ꦯꦸꦭ꧀ꦠ (Sulta)

• ꦯꦸꦭ꧀ꦠ (Sulta) + ꦤ (Na [Aksara Ngelegéna]) = ꦯꦸꦭ꧀ꦠꦤ (Sultana)

• ꦯꦸꦭ꧀ꦠꦤ (Sultana) + ꧀ꦡ (Pasangan Ta [Aksara Murda]) = ꦯꦸꦭ꧀ꦠꦤ꧀ꦡ (Sultanta)

• ꦯꦸꦭ꧀ꦠꦤ꧀ꦡ (Sultanta) + ꦽ (Sandhangan Re [Keret]) = ꦯꦸꦭ꧀ꦠꦤ꧀ꦡꦽ (Sultantre)

• ꦯꦸꦭ꧀ꦠꦤ꧀ꦡꦽ (Sultantre) + ꦁ (Sandhangan Ng [Cecak]) = ꦯꦸꦭ꧀ꦠꦤ꧀ꦡꦽꦁ (Sultantreng)

• ꦯꦸꦭ꧀ꦠꦤ꧀ꦡꦽꦁ (Sultantreng) + ꦒ (Ga [Aksara Ngelegéna]) + ꦤ (Na [Aksara Ngelegéna]) = ꦯꦸꦭ꧀ꦠꦤ꧀ꦡꦽꦁꦒꦤ (Sultan Trenggana)

Nomor 4

• ꦺ (Sandhangan É [Taling]) + ꦢ (Da [Aksara Ngelegéna]) = ꦢꦺ (Dé)

• ꦢꦺ (Dé) + ꦮ (Wa [Aksara Ngelegéna]) + ꦶ (Sandhangan I [Wulu]) = ꦢꦺꦮꦶ (Déwi)

• ꦢꦺꦮꦶ (Déwi) + ꦯ (Sa [Aksara Murda]) + ꦶ (Sandhangan I [Wulu]) = ꦢꦺꦮꦶꦯꦶ (Déwisi)

• ꦢꦺꦮꦶꦯꦶ (Déwisi) + ꦤ (Na [Aksara Ngelegéna]) + ꧀ꦠ (Pasangan Ta) = ꦢꦺꦮꦶꦯꦶꦤ꧀ꦠ (Déwi Sinta)

Nomor 5

• ꦄ (A [Aksara Swara]) + ꦫ (Ra [Aksara Ngelegéna]) = ꦄꦫ (Ara)

• ꦄꦫ (Ara) + ꦧ (Ba) = ꦄꦫꦧ (Araba)

• ꦄꦫꦧ (Araba) + ꧀ꦯ (Pasangan Sa [Aksara Murda]) = ꦄꦫꦧ꧀ꦯ (Arabsa)

• ꦄꦫꦧ꧀ꦯ (Arabsa) + ꦈ (U [Aksara Swara]) = ꦄꦫꦧ꧀ꦯꦈ (Arabsau)

• ꦄꦫꦧ꧀ꦯꦈ (Arabsau) + ꦢ (Da [Aksara Ngelegéna]) + ꦢ (Da [Aksara Ngelegéna]) + ꦶ (Sandhangan I [Wulu]) = ꦄꦫꦧ꧀ꦯꦈꦢꦶ (Arab Saudi)

Catatan:

Aksara Ngelegéna => Ha na ca ra ka, dst.

Aksara Swara => Aksara Vokal di Bahasa Jawa. Aksara Swara ada 5 huruf yaitu: A I U E O

Aksara Murda = Na Ka Ta Sa Pa Nya Ga dan Ba.

Sandhangan => adalah bunyi huruf yang mengubah huruf aslinya. Contoh: ꦫ (Ra), ditambah dengan sandhangan Wulu ( ꦶ ) yang berbunyi I dan merubah huruf ꦫ (Ra) menjadi ꦫꦶ (Ri).
