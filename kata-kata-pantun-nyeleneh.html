2. plis help jangan ngasal/nyeleneh mau di kumpul​


Persamaan kuadrat umum

ax² + bx + c = 0 dengan a ≠ 0

(1) x² = -x² + 3x

<=> x² + x² - 3x = 0

<=> 2x² - 3x = 0 → pers. Kuadrat

(2) x² - 2x = x² - 3

<=> x² - x² - 2x + 3 = 0

<=> -2x + 3 = 0 → bukan pers. Kuadrat

(3) 2x - 3 = 2x² - x + 1

<=> 2x² - x - 2x + 1 + 3 = 0

<=> 2x² - 3x + 4 = 0 → pers. Kuadrat

(4) 16 - x² = 2x² + 5

<=> 2x² + x² + 5 - 16 = 0

<=> 3x² - 11 = 0 → pers. Kuadrat

Kesimpulan

Pernyataan yang menyatakan persamaan kuadrat adalah 1, 3 dan 4

Yang merupakan persamaan kuadrat adalah 3.

Bukti:

1. x² = -x² + 3x

x² + x² - 3x = 0

2x² - 3x = 0 (salah, karena tidak ada konstantanya)

2. x² - 2x = x² - 3

x² - x² - 2x + 3 = 0

-2x + 3 = 0 (salah, karena tidak ada koefisien x²)

3. 2x - 3 = 2x² - x + 1

0 = 2x² - x - 2x + 1 + 3

0 = 2x² - 3x + 4 (benar)

4. 16 - x² = 2x² + 5

0 = 2x² + x² + 5 - 16

0 = 3x² - 11 (salah, karena tidak ada koefisien x)

16. bantuin sama caranya. point gede+jawaban tercerdas. nyeleneh? gue report​


Luas daerah yang diarsir adalah 96 cm²

PENDAHULUAN

Barisan adalah urutan atau susunan bilangan berdasarkan aturan aturan yang berlaku. Deret adalah jumlah keseluruhan bilangan yang memiliki aturan tertentu.

Barisan

➠ [tex] \boxed{ U_1,U_2,U_3,U_4,U_5,....,U_n } [/tex]

Deret

➠ [tex] \boxed{ U_1+U_2 + U_3+U_4+U_5+....+U_n } [/tex]

Barisan Aritmatika adalah barisan bilangan yang memiliki selisih yang sama tiap suku selanjutnya. Deret aritmatika adalah penjumlahan urutan bilangan yang memiliki selisih tiap suku yang sama

Barisan Geometri adalah barisan bilangan yang memiliki rasio yang sama tiap suku selanjutnya. Deret Geometri adalah penjumlahan urutan bilangan yang memiliki rasio yang sama.

Rumus rumus Barisan Aritmatika

➠ [tex] \boxed{ U_n = a + (n -1)b } [/tex]

➠ [tex] \boxed{ b = U_2 - U_1 } [/tex]

➠ [tex] \boxed{ a = U_1 } [/tex]

➠ [tex] \boxed{ U_n = S_n - S(n-1) } [/tex]

Rumus rumus Deret Aritmatika

➠ [tex] \boxed{ S_n = \dfrac{ n}{2 }(2a + (n-1)b)} [/tex]

➠ [tex] \boxed{ S_n = \dfrac{ n}{2 }(U_1 + U_n) } [/tex]

Rumus rumus Barisan Geometri

➠ [tex] \boxed{ U_n = a × r^{n-1} } [/tex]

➠ [tex] \boxed{ r = \dfrac{ U_2}{U_1 } } [/tex]

Rumus Deret Geometri

Untuk r > 1

➠ [tex] \boxed{ S_n = \dfrac{ a(r^{n} -1)}{ r-1} } [/tex]

Untuk r < 1

➠ [tex] \boxed{ S_n = \dfrac{ a(1 - r^{n} )}{ 1 -r} } [/tex]

Rumus Deret Geometri Tak Hingga

➠ [tex] \boxed{ S_∞ = \dfrac{a }{1 - r } } [/tex]

PEMBAHASAN

[tex]\bf Jawab : [/tex]

Luas pertama (a = U1)

[tex]\rm = 12 × 6 [/tex]

[tex]\rm = 72~cm² [/tex]

Luas kedua (U2)

[tex]\rm =6×3 [/tex]

[tex]\rm =18 [/tex]

Maka rasio nya adalah

[tex]\rm =\dfrac{ 18}{ 72} [/tex]

[tex]\rm = \dfrac{ 1}{4} [/tex]

Jumlah tak hingga

[tex]\rm S_∞= \dfrac{a }{1-r } [/tex]

[tex]\rm = \dfrac{72 }{1-\dfrac{ 1}{4}} [/tex]

[tex]\rm = \dfrac{72 }{\dfrac{ 3}{4}} [/tex]

[tex]\rm =72×\dfrac{ 4}{3} [/tex]

[tex]\rm =\dfrac{ 288}{3} [/tex]

[tex]\rm =96~cm² [/tex]

[tex]\\[/tex]

DETAIL JAWABAN Mapel : Matematika Kelas : XI - SMAKode Soal : 2Materi : BAB 7 - Barisan dan DeretKode Kategorisasi : 11.2.7Kata Kunci : Deret Konvergen, Tak hingga
